= 准备 7- 模式聚合和卷以进行过渡
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
过渡之前，您必须确保 7- 模式聚合和卷符合过渡条件，并在过渡之前执行一些手动步骤。例如，某些卷类型无法过渡，过渡前必须从 7- 模式系统中删除任何 32 位数据。
