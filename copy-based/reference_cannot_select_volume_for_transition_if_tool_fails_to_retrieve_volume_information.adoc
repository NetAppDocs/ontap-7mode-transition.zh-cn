= 如果工具失败，则无法选择要过渡的卷 以检索卷信息
:icons: font
:imagesdir: ../media/


[role="lead"]
尝试选择卷时，卷选择窗格会显示 `无法检索卷信息` 错误消息。如果控制器繁忙，通常会发生此错误。



== 临时解决策

等待控制器不繁忙，然后重新选择卷。
