= 在 Linux 上安装或卸载 7- 模式过渡工具 （仅限基于副本的过渡）
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用命令行界面在 Linux 系统上下载并安装 7- 模式过渡工具。如果需要，您可以卸载 7- 模式过渡工具。

您只能在 Linux 系统上执行基于副本的过渡。Linux 不支持无副本过渡。


IMPORTANT: Linux 不支持 7- 模式过渡工具 Web 界面。您必须使用命令行界面在 Linux 系统上执行基于副本的过渡任务。
