= 过渡挂载点使用 DMMP 设备名称的 LUN
:icons: font
:imagesdir: ../media/


[role="lead"]
过渡挂载点使用 DMMP 设备名称的 LUN 之前，必须将 DMMP 设备名称替换为相应的文件系统 UUID 编号。您必须执行特定步骤为转换阶段做准备，并且必须在过渡后在主机上重新挂载 DMMP 设备。您可以对 Red Hat Enterprise Linux （ RHEL ） 5 和 RHEL 6 执行相同的过程。
